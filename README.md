# Космическая Натальная Карта (Natal Chart SPA)

Интерактивное веб-приложение для расчета натальной астрологической карты с элементами геймификации (RPG-режимом) и ежедневными транзитами («Астрологической погодой»).

## 🚀 Возможности
- **Точные астрономические расчеты:** Вычисление позиций планет на основе кеплеровых элементов (JPL J2000) и Луны по формулам Жана Миуса.
- **Интерактивная карта (SVG):** Векторная натальная карта с автопозиционированием планет и предотвращением их наложения.
- **Два формата вывода:**
  - 🎮 **Игровой (RPG) режим (по умолчанию):** Превращает вашу натальную карту в карточку персонажа RPG. Рассчитывает ваш игровой класс (*Дипломатичный Стратег* и др.), распределяет характеристики силы стихий и выдает список жизненных квестов на основе активных домов.
  - 🌌 **Астрологический режим:** Классический подробный разбор стихий, ядра личности и фокуса жизненной энергии.
- **Астрологическая погода (Транзиты):** Возможность наложить карту сегодняшнего дня поверх вашей натальной карты, чтобы посмотреть влияние текущих планет.
- **Поиск городов:** Интеграция с OpenStreetMap Nominatim для автоподбора координат и часового пояса.
- **Печатная версия:** Специальные CSS-стили оптимизируют страницу для печати на бумаге.

## 🛠️ Как запустить локально
Просто откройте файл `index.html` в вашем веб-браузере. Для работы автопоиска городов рекомендуется запускать проект через локальный веб-сервер:
```bash
python -m http.server 8000
```
После чего откройте в браузере `http://localhost:8000`.

## 🌐 Развертывание на GitHub Pages
1. Создайте пустой публичный репозиторий на вашем GitHub.
2. Свяжите локальный репозиторий с удаленным и отправьте код:
   ```bash
   git remote add origin <ссылка-на-ваш-репозиторий>
   git branch -M main
   git push -u origin main
   ```
3. Перейдите в настройки созданного репозитория на GitHub: **Settings -> Pages**.
4. В разделе **Build and deployment** выберите в качестве источника ветку `main` (папка `/root`) и нажмите **Save**.
5. Через пару минут ваш сайт будет доступен по ссылке: `https://<ваш-логин>.github.io/<имя-репозитория>/`
